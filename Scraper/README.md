# Book Metadata Scraper

A Python-based scraper/API integration project that fetches metadata for books using the Google Books API.

## Features

- Fetch book title
- Author name
- Genre
- Summary/description
- Cover image URL
- Ratings
- Save data into JSON and CSV
- Duplicate detection

## Tech Stack

- Python
- Requests
- JSON
- CSV
